title: Update Product Invoice Endpoint
sidebarTitle: Attach Product Invoice
description:  The Update Invoice for a Product Transaction endpoint allows you to patch and update the invoice for a specific product transaction identified by the `hashid` parameter. The request should include the invoice file in the form data. The API responds with the updated transaction details, including the transaction's ID and the new invoice value.
openapi: patch /transactions/product-transactions/{hashid}/invoice/
